## ฟีเจอร์

| ฟีเจอร์ | รายละเอียด |
|---------|-----------|
| ค้นหาอัจฉริยะ | พิมพ์ชื่อ ประเภท หรือตำแหน่งเพื่อค้นหา |
| กรองข้อมูล | กรองตามประเภท พื้นที่จัดเก็บ ชนิดกระจก |
| แจ้งเตือนสต็อกต่ำ | ระบบเตือนเมื่อวัสดุใกล้หมด |
| ย้ายสต็อก | ย้ายวัสดุระหว่างตำแหน่งจัดเก็บ |

## ย้ายสต็อก

::: details วิธีย้ายวัสดุไปตำแหน่งอื่น
1. เข้าหน้ารายละเอียดวัสดุ
2. กดปุ่ม **ย้ายสต็อก**
3. เลือกตำแหน่งปลายทาง
4. ระบุจำนวนที่ต้องการย้าย
5. กดยืนยัน
:::

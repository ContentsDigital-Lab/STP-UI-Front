# ตั้งค่าระบบ

ปรับแต่งระบบให้เหมาะกับการใช้งาน

::: warning สิทธิ์การเข้าถึง
หน้าตั้งค่าส่วนใหญ่เข้าถึงได้เฉพาะ **Admin** และ **Manager** เท่านั้น
:::

## เมนูตั้งค่า

| เมนู | รายละเอียด | สิทธิ์ |
|------|-----------|--------|
| จัดการลูกค้า | เพิ่ม แก้ไข ลบข้อมูลลูกค้า กำหนดชื่อ ที่อยู่ เบอร์โทร ส่วนลดประจำ | Admin, Manager |
| ประเภทงาน | กำหนดประเภทงานที่โรงงานรับผลิต เช่น กระจกนิรภัย กระจกเทมเปอร์ | Admin, Manager |
| ตั้งค่าราคา | กำหนดราคาต่อหน่วยสำหรับแต่ละประเภทงานและวัสดุ | Admin, Manager |
| สติกเกอร์ | ออกแบบ Template สติกเกอร์สำหรับติดบนชิ้นงาน | Admin, Manager |
| จัดการผู้ใช้งาน | เพิ่ม แก้ไข ลบบัญชีผู้ใช้ กำหนดสิทธิ์ | Admin |
| การแจ้งเตือน | ตั้งค่าว่าต้องการรับแจ้งเตือนอะไรบ้าง | ทุกคน |

---

## จัดการลูกค้า

เพิ่มและแก้ไขข้อมูลลูกค้าที่โรงงานให้บริการ

**ข้อมูลที่บันทึกได้:**

| ฟิลด์ | รายละเอียด |
|-------|-----------|
| ชื่อลูกค้า | ชื่อบุคคลหรือบริษัท |
| ที่อยู่ | ที่อยู่สำหรับจัดส่ง |
| เบอร์โทรศัพท์ | เบอร์ติดต่อ |
| ส่วนลดประจำ (%) | ส่วนลดที่ให้กับลูกค้ารายนี้เสมอ |

::: details วิธีเพิ่มลูกค้าใหม่
1. เข้า **ตั้งค่า** → **ลูกค้า**
2. กดปุ่ม **+ เพิ่มลูกค้า**
3. กรอกข้อมูล: ชื่อ, ที่อยู่, เบอร์โทร, ส่วนลด
4. กด **บันทึก**
:::

---

## ประเภทงาน

กำหนดประเภทงานที่โรงงานรับผลิต ระบบนำไปใช้ในการคำนวณราคาและวางแผนการผลิต

**ข้อมูลที่บันทึกได้:**

| ฟิลด์ | รายละเอียด |
|-------|-----------|
| ชื่อประเภทงาน | เช่น "กระจกนิรภัย", "กระจกลามิเนต" |
| จำนวนกระจกดิบต่อชิ้น | กระจกดิบที่ต้องใช้ผลิต 1 ชิ้นงาน |

::: details วิธีเพิ่มประเภทงาน
1. เข้า **ตั้งค่า** → **ประเภทงาน**
2. กดปุ่ม **+ เพิ่มประเภทงาน**
3. กรอกชื่อและจำนวนกระจกดิบ
4. กด **บันทึก**
:::

---

## ตั้งค่าราคา

กำหนดราคาขายต่อหน่วยสำหรับงานแต่ละประเภท ระบบนำไปคำนวณราคาในบิลอัตโนมัติ

**รูปแบบราคา:**

| ประเภทราคา | รายละเอียด |
|-----------|-----------|
| ราคาต่อตารางเมตร | ราคาพื้นฐานคำนวณจากพื้นที่กระจก |
| บริการเพิ่มเติม | เช่น ขัดขอบ, เจาะรู, เจาะขอบ, ดัดโค้ง |

::: details วิธีกำหนดราคา
1. เข้า **ตั้งค่า** → **ราคา**
2. เลือกประเภทงานที่ต้องการตั้งราคา
3. กรอกราคาต่อตารางเมตรและราคาบริการเสริม
4. กด **บันทึก**
:::

---

## ออกแบบสติกเกอร์

สร้างและจัดการ Template สติกเกอร์สำหรับพิมพ์ติดชิ้นงาน

::: tip ดูคู่มือเพิ่มเติม
คู่มือการออกแบบสติกเกอร์โดยละเอียดอยู่ที่ [ออกแบบสติกเกอร์](/guide/sticker-designer)
:::

::: details วิธีสร้าง Template สติกเกอร์
1. เข้า **ตั้งค่า** → **สติกเกอร์**
2. กดปุ่ม **สร้าง Template ใหม่** หรือเลือกแก้ไข Template ที่มีอยู่
3. ใช้เครื่องมือแก้ไขเพื่อเพิ่ม: ข้อความ, รูปภาพ, QR Code, ตัวแปรข้อมูล
4. กดบันทึก
5. สามารถนำไปพิมพ์ได้จากหน้าสถานีงาน
:::

---

## จัดการผู้ใช้งาน

เพิ่ม แก้ไข หรือลบบัญชีผู้ใช้ และกำหนดสิทธิ์การเข้าถึง

**สิทธิ์ในระบบ:**

| ระดับ | รายละเอียด |
|-------|-----------|
| **Admin** | เข้าถึงได้ทุกส่วนของระบบ รวมถึงตั้งค่าและจัดการผู้ใช้ |
| **Manager** | จัดการออเดอร์, คลัง, ดูรายงาน ไม่สามารถจัดการผู้ใช้ |
| **Worker** | ทำงานที่สถานี เบิกวัสดุ แจ้งเคลม |

::: details วิธีเพิ่มผู้ใช้
1. เข้า **ตั้งค่า** → **ผู้ใช้งาน**
2. กดปุ่ม **เพิ่มผู้ใช้**
3. กรอกข้อมูล: ชื่อ, ชื่อผู้ใช้, รหัสผ่าน, ตำแหน่ง
4. เลือกสิทธิ์: **Admin** / **Manager** / **Worker**
5. กดบันทึก
:::

::: details วิธีเปลี่ยนรหัสผ่าน
1. เข้า **ตั้งค่า** → **ผู้ใช้งาน**
2. กดปุ่ม **แก้ไข** ที่ผู้ใช้ที่ต้องการ
3. กรอกรหัสผ่านใหม่
4. กดบันทึก
:::

## การแจ้งเตือน

กำหนดการแจ้งเตือนที่ต้องการรับ

| การแจ้งเตือน | รายละเอียด |
|-------------|-----------|
| สต็อกต่ำ | แจ้งเตือนเมื่อวัสดุในคลังต่ำกว่าจุดสั่งซื้อ |
| ออเดอร์ใหม่ | แจ้งเตือนเมื่อมีออเดอร์ใหม่เข้า |
| งานเสร็จ | แจ้งเตือนเมื่อชิ้นงานผ่านสถานีสุดท้าย |
| เสียง | เปิด/ปิดเสียงแจ้งเตือน |

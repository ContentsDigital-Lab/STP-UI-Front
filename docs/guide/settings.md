# ตั้งค่าระบบ

ปรับแต่งระบบให้เหมาะกับการใช้งาน

::: warning สิทธิ์การเข้าถึง
หน้าตั้งค่าส่วนใหญ่เข้าถึงได้เฉพาะ **Admin** เท่านั้น
:::

## เมนูตั้งค่า

| เมนู | รายละเอียด |
|------|-----------|
| จัดการลูกค้า | เพิ่ม แก้ไข ลบข้อมูลลูกค้า กำหนดชื่อ ที่อยู่ เบอร์โทร ส่วนลดประจำ |
| ประเภทงาน | กำหนดประเภทงานที่โรงงานรับผลิต เช่น กระจกนิรภัย กระจกเทมเปอร์ |
| ตั้งค่าราคา | กำหนดราคาต่อหน่วยสำหรับแต่ละประเภทงานและวัสดุ |
| สติกเกอร์ | ออกแบบ Template สติกเกอร์สำหรับติดบนชิ้นงาน |
| จัดการผู้ใช้งาน | เพิ่ม แก้ไข ลบบัญชีผู้ใช้ กำหนดสิทธิ์ |
| การแจ้งเตือน | ตั้งค่าว่าต้องการรับแจ้งเตือนอะไรบ้าง |

## ออกแบบสติกเกอร์

::: details วิธีสร้าง Template สติกเกอร์
1. เข้า **ตั้งค่า** → **สติกเกอร์**
2. กดปุ่ม **สร้าง Template ใหม่** หรือเลือกแก้ไข Template ที่มีอยู่
3. ใช้เครื่องมือแก้ไขเพื่อเพิ่ม: ข้อความ, รูปภาพ, QR Code
4. กดบันทึก
5. สามารถนำไปพิมพ์ได้จากหน้าสถานีงาน
:::

## เพิ่มผู้ใช้งานใหม่

::: details วิธีเพิ่มผู้ใช้
1. เข้า **ตั้งค่า** → **ผู้ใช้งาน**
2. กดปุ่ม **เพิ่มผู้ใช้**
3. กรอกข้อมูล: ชื่อ, ชื่อผู้ใช้, รหัสผ่าน, ตำแหน่ง
4. เลือกสิทธิ์: **Admin** / **Manager** / **Worker**
5. กดบันทึก
:::

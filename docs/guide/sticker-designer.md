# ออกแบบสติกเกอร์

<Card title="สติกเกอร์คืออะไร?" variant="blue">
สติกเกอร์คือป้ายที่พิมพ์ออกมาแล้วติดบนชิ้นงาน ประกอบด้วย QR Code, ข้อมูลชิ้นงาน, รหัสออเดอร์ ฯลฯ — ช่วยให้พนักงานสแกนติดตามงานได้ทุกสถานีโดยไม่ต้องค้นหาด้วยตัวเอง
</Card>

::: warning สิทธิ์การเข้าถึง
การออกแบบสติกเกอร์ทำได้เฉพาะ **Admin** และ **Manager** เท่านั้น พนักงานทุกคนสามารถ **พิมพ์** สติกเกอร์ได้จากสถานีที่มีปุ่มพิมพ์สติกเกอร์
:::

## องค์ประกอบทั้งหมดในสติกเกอร์

สติกเกอร์สร้างจากการวางองค์ประกอบซ้อนกันบนพื้นที่ขาว สามารถผสมกันได้อย่างอิสระ

---

### ✏️ ข้อความ (Text)

แสดงข้อความที่ไม่เปลี่ยนแปลง เช่น ชื่อโรงงาน คำว่า "กระจกนิรภัย" หรือข้อความเตือนที่แสดงในสติกเกอร์ทุกใบ

**วิธีใช้:** กดเพิ่มองค์ประกอบ "ข้อความ" แล้วพิมพ์ข้อความที่ต้องการในช่องแก้ไขด้านขวา

| ค่าตั้งค่า | รายละเอียด |
|-----------|-----------|
| ข้อความ | พิมพ์ข้อความที่ต้องการแสดง ทุกใบพิมพ์จะเหมือนกันหมด |
| ขนาดตัวอักษร | ตัวเลขยิ่งมาก ตัวหนังสือยิ่งใหญ่ |
| ตัวหนา | เปิด = ตัวหนา ปิด = ตัวบาง |
| ตัวเอียง | เปิด = ตัวเอียง |
| สีตัวอักษร | เลือกสีตัวหนังสือ |
| แบบอักษร | เลือกฟอนต์ที่ต้องการ |

::: tip เมื่อไหรควรใช้ข้อความ
ใช้สำหรับข้อมูลที่เหมือนกันทุกใบ เช่น ชื่อโรงงาน ที่อยู่ คำเตือน หรือชื่อคอลัมน์ เช่น "ลูกค้า:" ก่อนหน้าองค์ประกอบข้อมูลตัวแปร
:::

---

### 🔄 ข้อมูลอัตโนมัติ (Dynamic)

แสดงข้อมูลที่ดึงมาจากออเดอร์หรือชิ้นงานจริงขณะพิมพ์ — แต่ละใบจะแสดงข้อมูลของชิ้นงานนั้นโดยเฉพาะ เช่น ชื่อลูกค้า ขนาดกระจก หรือรหัสออเดอร์

**วิธีใช้:** กดเพิ่มองค์ประกอบ "ข้อมูลอัตโนมัติ" แล้วเลือกตัวแปรที่ต้องการจากรายการ หรือพิมพ์ตัวแปรในช่องข้อความโดยใช้รูปแบบ `{{ชื่อตัวแปร}}` เช่น `{{customerName}}`

**ตัวอย่าง:** ถ้าตั้งเป็น `ออเดอร์: {{orderCode}}` ใบที่พิมพ์สำหรับออเดอร์ A001 จะแสดง `ออเดอร์: A001` และใบของออเดอร์ A002 จะแสดง `ออเดอร์: A002` โดยอัตโนมัติ

| ค่าตั้งค่า | รายละเอียด |
|-----------|-----------|
| ข้อความ | พิมพ์ข้อความ แล้วแทรกตัวแปรในรูปแบบ `{{ชื่อตัวแปร}}` ได้เลย เช่น `ขนาด: {{dimensions}} มม.` |
| ขนาดตัวอักษร | กำหนดขนาด |
| ตัวหนา / ตัวเอียง | เปิด/ปิด |
| สีตัวอักษร | เลือกสี |
| แบบอักษร | เลือกฟอนต์ |

**ตัวแปรทั้งหมดที่ใช้ได้:**

#### ข้อมูลชิ้นงาน

| ตัวแปร | แสดงอะไร | ตัวอย่าง |
|--------|---------|---------|
| `{{paneNumber}}` | หมายเลขชิ้นงาน | 1, 2, 3... |
| `{{paneId}}` | รหัสชิ้นงานในระบบ | 6745abc... |
| `{{glassType}}` | ประเภทกระจก | กระจกนิรภัย |
| `{{dimensions}}` | ขนาดครบชุด กว้างxสูงxหนา | 600x900x6 |
| `{{width}}` | ความกว้าง (มิลลิเมตร) | 600 |
| `{{height}}` | ความสูง (มิลลิเมตร) | 900 |
| `{{thickness}}` | ความหนา (มิลลิเมตร) | 6 |
| `{{qrCode}}` | รหัสที่เข้ารหัสลงใน QR Code (ใช้คู่กับองค์ประกอบ QR Code) | |
| `{{status}}` | สถานะชิ้นงานปัจจุบัน | รอดำเนินการ |
| `{{assignedTo}}` | ชื่อพนักงานที่รับผิดชอบ | สมชาย |

#### ข้อมูลออเดอร์

| ตัวแปร | แสดงอะไร | ตัวอย่าง |
|--------|---------|---------|
| `{{orderCode}}` | รหัสออเดอร์ | ORD-2024-001 |
| `{{customerName}}` | ชื่อลูกค้า | ห้างสรรพสินค้าเซ็นทรัล |
| `{{materialName}}` | ชื่อวัสดุหรือกระจก | กระจกใส 6mm |
| `{{quantity}}` | จำนวนชิ้นทั้งหมดในออเดอร์ | 15 |

#### วันที่และเวลา

| ตัวแปร | แสดงอะไร | ตัวอย่าง |
|--------|---------|---------|
| `{{date}}` | วันที่พิมพ์สติกเกอร์ (รูปแบบไทย) | 30 มีนาคม 2567 |
| `{{time}}` | เวลาที่พิมพ์ (รูปแบบไทย) | 14:35 น. |

### 📱 QR Code

แสดง QR Code ที่เมื่อสแกนด้วยกล้องโทรศัพท์หรือเครื่องสแกน จะนำไปยังข้อมูลชิ้นงานนั้นในระบบ — พนักงานในสถานีถัดไปสแกน QR นี้เพื่อรับงานได้ทันทีโดยไม่ต้องค้นหาด้วยตัวเอง

**วิธีใช้:** กดเพิ่มองค์ประกอบ "QR Code" แล้ววางในตำแหน่งที่ต้องการ ปรับขนาดให้เหมาะกับพื้นที่

| ค่าตั้งค่า | รายละเอียด |
|-----------|-----------|
| ความกว้าง | ขนาดกว้างของ QR Code บนสติกเกอร์ |
| ความสูง | ขนาดสูงของ QR Code บนสติกเกอร์ (ควรเท่ากับความกว้างเพื่อให้เป็นสี่เหลี่ยมจตุรัส) |
| ค่าที่เข้ารหัส | ใช้ตัวแปร `{{qrCode}}` เพื่อให้ระบบใส่รหัสชิ้นงานให้อัตโนมัติ หรือพิมพ์ค่าตายตัวถ้าต้องการ QR เดิมทุกใบ |

::: tip ขนาด QR ที่แนะนำ
QR Code ควรมีขนาดอย่างน้อย 20x20 มิลลิเมตรเพื่อให้สแกนได้ง่าย ถ้าเล็กเกินไปอาจสแกนไม่ได้
:::

### 🔷 รูปร่าง (Shape)

วาดรูปร่างต่าง ๆ เพื่อตกแต่ง แบ่งพื้นที่ หรือสร้างกล่องล้อมข้อมูล สามารถกำหนดสีพื้น สีขอบ และความหนาขอบได้

**วิธีใช้:** กดเพิ่มองค์ประกอบ "รูปร่าง" แล้วเลือกรูปแบบ วางในตำแหน่งที่ต้องการ ปรับขนาดและสีที่แผงด้านขวา

#### รูปร่างที่ใช้ได้

**สี่เหลี่ยม (Rectangle)**
รูปสี่เหลี่ยมผืนผ้าหรือสี่เหลี่ยมจตุรัส ใช้สร้างกล่องล้อมรอบข้อมูลหรือพื้นที่เน้นสี

| ค่าตั้งค่า | รายละเอียด |
|-----------|-----------|
| ความกว้าง | ขนาดกว้าง |
| ความสูง | ขนาดสูง |
| สีพื้น | สีที่เติมข้างใน ปิด = โปร่งใส |
| สีขอบ | สีของเส้นขอบ |
| ความหนาขอบ | ความหนาของเส้นขอบ |
| ป้ายกำกับ | ข้อความที่แสดงตรงกลางรูปสี่เหลี่ยม |
| สีตัวอักษรป้าย | สีข้อความในกล่อง |

**วงกลม (Circle)**
รูปวงกลมสมบูรณ์ ใช้เน้นข้อมูลหรือสร้างตราประทับ

**วงรี (Ellipse)**
รูปวงรีที่ปรับความกว้างและสูงต่างกันได้

**สามเหลี่ยม (Triangle)**
รูปสามเหลี่ยม ใช้สร้างลูกศรหรือสัญลักษณ์

**ดาว (Star)**
รูปดาว ใช้เป็นสัญลักษณ์เน้นหรือตกแต่ง

### ➖ เส้น (Line)

วาดเส้นตรงเพื่อแบ่งพื้นที่หรือตกแต่ง เช่น เส้นใต้ชื่อโรงงาน หรือเส้นแบ่งระหว่างข้อมูลชิ้นงานกับ QR Code

**วิธีใช้:** กดเพิ่มองค์ประกอบ "เส้น" แล้ววางในตำแหน่งที่ต้องการ ลากจุดปลายเพื่อเปลี่ยนทิศทางและความยาว

| ค่าตั้งค่า | รายละเอียด |
|-----------|-----------|
| สีเส้น | เลือกสีของเส้น |
| ความหนา | ตัวเลขยิ่งมาก เส้นยิ่งหนา |

---

### 🖼️ รูปภาพ (Image)

นำเข้ารูปภาพมาวางบนสติกเกอร์ เช่น โลโก้บริษัท รูปสัญลักษณ์ หรือลายน้ำ

**วิธีใช้:** กดเพิ่มองค์ประกอบ "รูปภาพ" แล้วอัปโหลดไฟล์รูปจากคอมพิวเตอร์ ปรับตำแหน่งและขนาดตามต้องการ

| ค่าตั้งค่า | รายละเอียด |
|-----------|-----------|
| ความกว้าง | ขนาดที่แสดงบนสติกเกอร์ |
| ความสูง | ขนาดที่แสดงบนสติกเกอร์ |
| ครอบตัดรูป | ปรับมุมมองรูปที่แสดง ถ้ารูปต้นฉบับกับพื้นที่ไม่ตรงกัน |

**รูปแบบไฟล์ที่รองรับ:** JPG, PNG, GIF, SVG

::: tip โลโก้โรงงาน
อัปโหลดโลโก้โรงงานครั้งเดียว แล้วนำไปวางในทุกแบบสติกเกอร์ ขนาดไฟล์ควรไม่เกิน 2MB และควรใช้ไฟล์ PNG พื้นหลังโปร่งใส เพื่อให้โลโก้ดูดีบนสติกเกอร์
:::

## เทคนิคการออกแบบสติกเกอร์

::: tip สติกเกอร์ที่ดีควรมี
1. **QR Code** — ใหญ่พอสแกนได้ง่าย วางในตำแหน่งที่เห็นชัด
2. **รหัสออเดอร์และชื่อลูกค้า** — เพื่อระบุได้ด้วยตาแม้ไม่สแกน
3. **ขนาดกระจก** — ช่วยให้พนักงานตรวจสอบได้ทันที
4. **หมายเลขชิ้นงาน** — เช่น ชิ้นที่ 1/5 เพื่อนับง่าย
:::

::: tip ขนาดสติกเกอร์ที่นิยม
- **50x30 mm** — สำหรับชิ้นงานเล็ก มีแค่ QR และรหัส
- **80x50 mm** — มาตรฐาน มีข้อมูลครบ
- **100x70 mm** — สำหรับชิ้นงานใหญ่ มีรายละเอียดมาก
:::

</div>
</Steps>

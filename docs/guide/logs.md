# ประวัติการทำรายการ

<Card title="หน้าประวัติคืออะไร?" variant="blue">
หน้าประวัติแสดงการเคลื่อนไหวของวัสดุทั้งหมด — วัสดุเข้า, เบิกออก, ย้าย, เคลม, ตัด — ช่วยให้ตรวจสอบได้ว่าวัสดุแต่ละชิ้นไปอยู่ที่ไหนบ้าง
</Card>

## ข้อมูลที่แสดง

แต่ละรายการในประวัติแสดง:

| ข้อมูล | รายละเอียด |
|--------|-----------|
| วัสดุ | ชื่อ ประเภท ความหนา สี ของวัสดุนั้น |
| ประเภทการทำรายการ | ดูหัวข้อถัดไป |
| จำนวน | บวก = เพิ่มสต็อก, ลบ = ลดสต็อก |
| ตำแหน่ง | ที่อยู่ในคลัง (ถ้ามี) |
| ออเดอร์ที่เกี่ยวข้อง | รหัสออเดอร์ |
| ชิ้นงานที่เกี่ยวข้อง | หมายเลขชิ้นงาน (Pane) |
| พนักงาน | ชื่อผู้ทำรายการ |
| วันที่/เวลา | เวลาที่เกิดรายการ |

## ประเภทการทำรายการ

| ประเภท | ความหมาย |
|--------|---------|
| **นำเข้า** | รับวัสดุเข้าคลัง เช่น รับกระจกใหม่จากซัพพลายเออร์ |
| **ย้ายนำเข้า** | ย้ายวัสดุระหว่างตำแหน่งในคลัง (ฝั่งปลายทาง) |
| **เบิกออก** | เบิกวัสดุออกเพื่อใช้ในออเดอร์ |
| **ย้ายออก** | ย้ายวัสดุออกจากตำแหน่ง (ฝั่งต้นทาง) |
| **เคลม** | วัสดุเสียหายหรือมีปัญหา |
| **ตัด** | วัสดุถูกนำไปตัด/ใช้ในการผลิต |

## การค้นหาและกรอง

สามารถกรองรายการได้ตาม:

- **วัสดุ** — ค้นหาชื่อวัสดุ
- **ประเภทการทำรายการ** — เลือกประเภทที่ต้องการดู
- **ช่วงวันที่** — กำหนดวันเริ่มต้นและสิ้นสุด
- **พนักงาน** — ดูรายการของพนักงานคนหนึ่ง

## อัปเดตแบบเรียลไทม์

::: tip
รายการในหน้าประวัติอัปเดตอัตโนมัติเมื่อมีการทำรายการใหม่ — ไม่ต้องกดรีเฟรช
:::

## ความสัมพันธ์กับส่วนอื่น

| การทำงาน | สร้างรายการประวัติประเภท |
|---------|----------------------|
| นำเข้าวัสดุจากหน้าคลัง | นำเข้า |
| ย้ายวัสดุในคลัง | ย้ายนำเข้า + ย้ายออก |
| เบิกวัสดุจากสถานี | เบิกออก |
| แจ้งเคลมชิ้นงาน | เคลม |

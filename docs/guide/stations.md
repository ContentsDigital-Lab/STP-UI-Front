# การทำงานที่สถานี

สถานีงานคือจุดทำงานในโรงงาน เช่น สถานีตัดกระจก, สถานีขัดขอบ ฯลฯ

## ขั้นตอนการทำงาน

<Flow :items="stationFlow" />

## วิธีใช้งาน

<Steps>
<div class="step" data-step="1">
<h4>เช็คอิน</h4>
<p>เลือกสถานีที่ต้องการทำงาน แล้วกดปุ่ม <strong>เช็คอิน</strong> เพื่อลงชื่อเข้าสถานี</p>
</div>
<div class="step" data-step="2">
<h4>สแกน QR Code</h4>
<p>กดปุ่ม <strong>สแกน QR</strong> แล้วใช้กล้องสแกน QR Code ที่ติดอยู่บนชิ้นงาน — ระบบจะแสดงข้อมูลชิ้นงาน</p>
</div>
<div class="step" data-step="3">
<h4>เริ่มทำงาน</h4>
<p>ตรวจสอบข้อมูลชิ้นงาน (ประเภทกระจก, ขนาด, กระบวนการ) แล้วกดปุ่ม <strong>เริ่มทำงาน</strong></p>
</div>
<div class="step" data-step="4">
<h4>เสร็จสิ้น</h4>
<p>เมื่อทำเสร็จ กดปุ่ม <strong>เสร็จสิ้น</strong></p>
</div>
<div class="step" data-step="5">
<h4>สแกนออก</h4>
<p>สแกน QR Code อีกครั้งเพื่อส่งออกจากสถานี — ชิ้นงานจะถูกส่งต่อไปสถานีถัดไปอัตโนมัติ</p>
</div>
</Steps>

## คิวงานในสถานี

ที่หน้าสถานีจะแสดงข้อมูล 3 ส่วน:

- **งานที่รอ** — ชิ้นงานที่รอเข้าสถานีนี้
- **งานที่กำลังทำ** — ชิ้นงานที่กำลังอยู่ในสถานี
- **ประวัติ** — ชิ้นงานที่ผ่านสถานีนี้ไปแล้ว

::: tip อัปเดตอัตโนมัติ
ข้อมูลที่หน้าสถานีจะอัปเดตแบบ **เรียลไทม์** — ไม่ต้องกดรีเฟรชหน้าจอ เมื่อมีชิ้นงานใหม่เข้ามาหรือสถานะเปลี่ยน จะเห็นทันที
:::

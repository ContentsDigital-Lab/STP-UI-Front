# หน้าแดชบอร์ด

หน้าแรกที่คุณจะเห็นหลังเข้าสู่ระบบ แสดงภาพรวมของงานทั้งหมดในที่เดียว

## ข้อมูลสรุป

ด้านบนของหน้าแดชบอร์ดจะแสดงตัวเลขสรุป 4 รายการ:

<Card title="สต็อกทั้งหมด" variant="blue">จำนวนวัสดุทั้งหมดที่มีในคลัง</Card>
<Card title="แจ้งเตือนสต็อกต่ำ" variant="red">จำนวนวัสดุที่ใกล้หมดและต้องสั่งซื้อเพิ่ม</Card>
<Card title="คำขอที่รอนุมัติ" variant="orange">ออเดอร์ที่ยังไม่ได้ดำเนินการ</Card>
<Card title="ทำรายการวันนี้" variant="green">จำนวนรายการที่ทำเสร็จแล้ววันนี้</Card>

## กราฟและรายงาน

- **กระแสการหมุนเวียนวัสดุ** — กราฟแสดงปริมาณการนำเข้าและเบิกจ่ายวัสดุในแต่ละช่วงเวลา
- **กิจกรรมล่าสุด** — รายการกิจกรรมที่เพิ่งเกิดขึ้นในระบบ เช่น เบิกวัสดุ, สแกนชิ้นงาน, รับออเดอร์

## เมนูด่วน

ที่หน้าแดชบอร์ดมีปุ่มลัดสำหรับเข้าถึงฟีเจอร์ที่ใช้บ่อย เช่น รับออเดอร์ ดูสต็อก เป็นต้น

## การแจ้งเตือน

กดไอคอน **กระดิ่ง** ที่มุมขวาบน เพื่อดูการแจ้งเตือนต่าง ๆ:

::: details ดูรายการแจ้งเตือนที่ระบบส่งได้
- มีออเดอร์ใหม่เข้ามา
- ชิ้นงานเข้าสถานีงาน
- สต็อกวัสดุต่ำกว่าที่กำหนด
- มีการเคลมสินค้า
- งานผลิตเสร็จสมบูรณ์
:::

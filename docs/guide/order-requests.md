# สร้างคำสั่งซื้อ

คำสั่งซื้อคือจุดเริ่มต้นของงานทุกอย่างในระบบ เมื่อลูกค้าสั่งทำกระจก คุณจะเริ่มที่นี่

::: warning สิทธิ์การเข้าถึง
เฉพาะ **Admin** และ **Manager** เท่านั้นที่สามารถสร้างและจัดการคำสั่งซื้อได้
:::

## วิธีสร้างคำสั่งซื้อ

<Steps>
<div class="step" data-step="1">
<h4>เปิดหน้าคำสั่งซื้อ</h4>
<p>กดเมนู <strong>คำสั่งซื้อ</strong> ที่แถบด้านซ้าย แล้วกดปุ่ม <strong>สร้างคำสั่งซื้อ</strong></p>
</div>
<div class="step" data-step="2">
<h4>เลือกลูกค้า</h4>
<p>เลือกจากรายชื่อลูกค้าที่มีอยู่แล้ว หรือกด <strong>"เพิ่มลูกค้าใหม่"</strong> ถ้ายังไม่มีในระบบ</p>
</div>
<div class="step" data-step="3">

<p>ระบุประเภทงาน (เช่น กระจกนิรภัย, กระจกเทมเปอร์), จำนวน และราคาประเมิน</p>
</div>
<div class="step" data-step="4">
<h4>กำหนดการจัดส่ง</h4>

</div>
<div class="step" data-step="5">
<h4>มอบหมายผู้รับผิดชอบ</h4>

</div>
<div class="step" data-step="6">
<h4>บันทึก</h4>
<p>กดปุ่ม <strong>บันทึก</strong> — ระบบจะสร้างรายการผลิตและชิ้นงานให้โดยอัตโนมัติ</p>
</div>
</Steps>

## จัดการลูกค้า

<Card title="เพิ่ม / แก้ไข / ลบ ข้อมูลลูกค้า" variant="blue">

- **เพิ่มลูกค้าใหม่** — กรอกชื่อ, ที่อยู่, เบอร์โทร, ส่วนลดประจำ
- **แก้ไขข้อมูล** — อัปเดตที่อยู่ เบอร์โทร หรือส่วนลด
- **ลบลูกค้า** — ลบข้อมูลลูกค้าที่ไม่ใช้แล้ว

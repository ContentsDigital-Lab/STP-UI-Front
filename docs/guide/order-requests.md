# รับออเดอร์

ออเดอร์คือจุดเริ่มต้นของงานทุกอย่างในระบบ เมื่อลูกค้าสั่งทำกระจก คุณจะเริ่มที่นี่

::: warning สิทธิ์การเข้าถึง
เฉพาะ **Admin** และ **Manager** หรือบทบาทที่ได้รับสิทธิ์ `orders:create` เท่านั้นที่สามารถรับและจัดการออเดอร์ได้
:::

# เบิกวัสดุ

เมื่อต้องการใช้วัสดุจากคลังในการผลิต ให้ทำการเบิกผ่านระบบ

## ขั้นตอนการเบิก

<Steps>
<div class="step" data-step="1">
<h4>เปิดหน้าเบิกวัสดุ</h4>
<p>กดเมนู <strong>เบิกวัสดุ</strong> ที่แถบด้านซ้าย แล้วกดปุ่ม <strong>เบิกวัสดุ</strong></p>
</div>
<div class="step" data-step="2">
<h4>เลือกคำสั่งซื้อ</h4>
<p>เลือกคำสั่งซื้อที่ต้องการเบิกวัสดุให้</p>
</div>
<div class="step" data-step="3">
<h4>เลือกวัสดุ</h4>
<p>เลือกวัสดุจากรายการในคลัง</p>
</div>
<div class="step" data-step="4">
<h4>เลือกประเภทสต็อก</h4>
<p><strong>ของใหม่ (Raw)</strong> — วัสดุที่ยังไม่ได้ใช้ หรือ <strong>ของเหลือใช้ (Reuse)</strong> — วัสดุที่เหลือจากงานก่อนหน้า</p>
</div>
<div class="step" data-step="5">
<h4>ระบุจำนวน</h4>
<p>ใส่จำนวนที่ต้องการเบิก</p>
</div>
<div class="step" data-step="6">
<h4>ยืนยัน</h4>
<p>กดปุ่ม <strong>ยืนยัน</strong> — ระบบจะตัดสต็อกให้อัตโนมัติ</p>
</div>
</Steps>

::: danger ข้อควรระวัง
เมื่อเบิกวัสดุแล้ว ระบบจะ **ตัดสต็อกทันที** กรุณาตรวจสอบจำนวนให้ถูกต้องก่อนกดยืนยัน
:::
